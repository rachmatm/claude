# Project Modes

## Bootstrap Mode

### Auto Detection

If:

* requirements.pdf exists
* requirements.docx exists
* assignment.pdf exists
* client-brief.md exists

AND

* docs/product-requirements.md does not exist

Then:

Suggest running Bootstrap Mode.

Example:

"I found a requirements document but project documentation has not been initialized.

Would you like me to execute Bootstrap Mode and generate:

* product-requirements.md
* user-journey.md
* acceptance-criteria.md
* test-plan.md
* tech-stack.md
* architecture.md
* development-status.md
* current-sprint.md
* project-memory.md"

---

### Bootstrap Actions

1. Analyze requirements.
2. Extract requirements.
3. Generate project documentation.
4. Create implementation roadmap.
5. Initialize sprint tracking.
6. Initialize project memory.
